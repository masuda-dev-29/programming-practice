# Household GUI App

## 概要

PythonのGUIライブラリ「Tkinter」を用いて家計簿アプリを制作する学習プロジェクトです。

本プロジェクトの目的は、単に家計簿アプリを完成させることではなく、GUIプログラミングの基礎から設計・実装までを段階的に学習することです。

また、Git/GitHubによるバージョン管理を行い、学習過程を履歴として残すことも目的としています。

---

# 学習目標

- PythonによるGUIプログラミングを理解する
- Tkinterの基本部品（Widget）の使い方を習得する
- イベント駆動プログラミングを理解する
- CSVを利用したデータ保存を学ぶ
- Treeviewによる一覧表示を実装する
- 家計簿アプリを完成させる
- Git/GitHubを利用した開発フローに慣れる

# 学習スタイル

本プロジェクトでは完成コードを写すのではなく、段階的に理解を深めながら実装する。

各Lessonは次の流れで進行する。

1. 講義
2. サンプルコードの解説
3. ドリル（問題）
4. 解説
5. 小課題

## プロジェクト方針

- Python初学者向けに進める
- GUI開発未経験を前提とする
- 一度に多くの内容を扱わず、1Lessonにつき1テーマを学ぶ
- 必ず理解を確認するドリルを実施する
- 「講義 → ドリル → 解説 → 小課題」の順で進行する
- 必要に応じて前回内容を復習する
- 完成コードを最初から提示しない
- 学習者が自分で考える時間を確保する
- GUI部品の役割や内部動作も解説する
- 実務的な書き方や設計も少しずつ紹介する
